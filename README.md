## Text Editor Project
Small text editor written in C, derived from kilo, following [this tutorial](https://viewsourcecode.org/snaptoken/kilo/index.html) from SnapToken on viewsourcecode.org

To compile the project, just download *kilo.c* and *Makefile*, and type **make** in your terminal.
Then just launch the *kilo* executable (you can pass a file name as an argument in order to open it).

The text editor is for the moment just a file viewer, with a working cursor.
